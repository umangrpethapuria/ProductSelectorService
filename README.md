# Sky product selector service

The service provides backend interface to allow selection of products based on customers location.

### Prerequisites


```
Java 8
Maven
```

### Installing

## Running the tests

```
cd scripts
./executeTests.sh
```


## Build package

```
cd scripts
./build.sh
```


## Launch

```
cd scripts
./launch.sh
```
